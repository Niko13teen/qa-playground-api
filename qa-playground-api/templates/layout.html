Description: POST: Sends data to the server to create a new object. Often used when uploading a file or submitting a web form.

Params:

"email":"test@gmail.com"

"first_name":"test"

"last_name":"test"


Content-Type: application/json

Response: 201

Description: DELETE request deletes the specified resource
